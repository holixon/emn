# Event Modeling Notation

[![incubating](https://img.shields.io/badge/lifecycle-INCUBATING-orange.svg)](https://github.com/holisticon#open-source-lifecycle)
[![sponsored](https://img.shields.io/badge/sponsoredBy-Holisticon-RED.svg)](https://holisticon.de/)

Event Modeling Notation (EMN) is a language specification for Event Modeling to be used in [bpmn.io](https://github.com/bpmn-io) based tool stack.
It provides a [M2 Model](https://en.wikipedia.org/wiki/Metamodeling) for [event modelling](https://eventmodeling.org/posts/what-is-event-modeling/). 

This repository is a source of the specification documentation site.
